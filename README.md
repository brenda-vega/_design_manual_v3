renda Vega — Design Manual v3

Sistema de diseño orientado a UI/UX + Product Thinking para creación de interfaces consistentes y escalables.

⸻

Scope

* Visual foundations (color, typography, spacing)
* Design tokens
* Figma naming system
* Layout (grid, breakpoints)
* Motion tokens
* UI components
* Accessibility (WCAG 2.2)
* Validation & handoff

⸻

Core Principle
No visual element exists without function.

Decision Model
Change [X] → to achieve [Y] → because user [Z]

Naming Convention
Category/Component/Variant

Examples
Button/Primary/Default
Input/Error
color/text/primary
spacing/16

⸻

Token System

* Spacing: base 8px scale
* Typography: semantic scale (display → body → label)
* Colors: semantic + brand + states
* Radius: 0–pill
* Shadows: elevation system

⸻

Layout

* Grid-based system
* Responsive breakpoints
* Consistent spacing rules

⸻

Motion

* Defined durations & easing
* Functional transitions only

⸻

Accessibility

* WCAG 2.2 compliant
* Contrast, sizing, touch targets

⸻

Workflow
Problem → User → Decision → UI → Validation → Handoff

⸻

Usage

1. Define problem
2. Apply decision model
3. Design using tokens
4. Follow naming system
5. Validate before delivery

Version

v3.0 — evolving system
